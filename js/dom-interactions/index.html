JavaScript > DOM interactions

Assume you have jQuery or Prototype, write performance-wise script to:

hide all even rows
Set 'visibility: hidden' on class name 'even' $(function() { $("ul .even").css("visibility", "hidden"); });
do same assuming there are no class names
use ':odd' since elements are selected using a zero-index $(function() { $("ul :odd").css("visibility", "hidden"); });
Build two arrays containing odd and even rows respectively.
Push odd and even rows into different arrays, then output in console $(function() { // Odd Rows var oddRows = []; $('ul :even').each(function (i, e) { oddRows.push($(e).text()); }); console.log("Odd Rows"); $.each( oddRows, function( key, value ) { console.log( key + ": " + value ); }); // Even Rows var evenRows = []; $('ul :odd').each(function (i, e) { evenRows.push($(e).text()); }); console.log("Even Rows"); $.each( evenRows, function( key, value ) { console.log( key + ": " + value ); }); });
Rearrange nodes in reverse order
Go though each li and preprend it to the parent node (this gives reverse order) $(function() { $('ul > li').each(function() { $(this).prependTo(this.parentNode); }); });

Ability to demonstrate few possible ways to accomplish this is a plus.
